# Sistema financeiro

Esse é um projeto para aprendizado de analise de dados e Javascript. Esse projeto existe somente para fins de estudos.

## Dependencias

Esse projeto usa o gerenciador de pacotes e o runtime [bun](https://bun.com/), então tenha certeza de ter ele instalado em sua máquina antes de começar.

A tecnologia usada no banco de dados é o [PostgreSQL](https://www.postgresql.org/), então também utilize um docker ou tenha ele instalado em sua maquina.
