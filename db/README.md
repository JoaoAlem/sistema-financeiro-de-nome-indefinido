# Banco de dados

Essa é a pasta que irá representar o projeto do banco de dados. Todas as modificações no banco de dados devem ser incluidas no arquivo `src/schema.ts`.

## Como começar?

1. Execute o comando: `bun install` para instalar as dependencias;
2. Copie o arquivo `.env.example` para `.env` e configure a URL do banco de dado;
3. Execute o comando `bun run db:migrate`;
4. Seja Feliz :tada:.
